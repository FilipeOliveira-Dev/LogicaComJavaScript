JAVASCRIPT

- Nesse dia, aprendemos mais sobre o comando if e operações booleanas.
- Recebemos uma instrução em "portugol", e fizemos o código para que ele imprima as informações de maneira correta, que faça sentido e sem erros.

- Nesse dia, aprendemos a trabalhar com captura de inputs e como armazenar essas informações em variáveis para usá-las durante a execução do seu programa.
- O usuário reponde 3 perguntas (nome, idade, linguagem de programação favorita), e o sistema exibe uma mensagem personalizada na tela com as respostas do usuário.

- Nesse dia, o desafio é criar os destinos possíveis de um jogo, em que o usuário consiga escolher, em qual carreira ele pretende seguir (Front-End ou Back-End).
- E após escolher a carreira, ele decide se quer se especializar, e no final ele pode colocar quais linguagens de programação ele deseja estudar. Ao final, o sistema deverá mostrar todas as tecnologias que o usuário digitou.

- Nesse dia, o desafio é criar um programa que gera um número aleatório entre 0 e 10, e o usuário tentará advinhar esse número.
- O programa dá 3 tentativas, e vai informando se o número secreto é maior ou menor do que ele digitou. O programa vai mostrando quantas tentativas o usuário ainda tem, e se não acertar, o programa mostra qual é o número secreto.

- Nesse dia 05, o desafio é criar um programa que cria uma lista de compras, com itens e suas respectivas categorias. O programa vai mostrando a lista atualizada separados por categoria.
- Nesse dia 06, o desafio é criar a opção de remover algum item da lista. Caso positivo, o programa imprime os elementos da lista atual, e a pessoa escreve qual deles deseja remover.
- O programa remove e imprime a confirmação. Se não for encontrado o item na lista, o programa mostra uma mensagem avisando.

- Nesse dia, o desafio é criar uma calculadora simples. Primeiro, o usuário escolhe uma opção de operação. Depois, o usuário insere os dois valores que deseja utilizar, e o programa imprimirá o resultado da operação.
- As opções disponíveis são: soma, subtração, multiplicação, divisão, e sair. Nessa última, o programa deverá parar de ser executado, mostrando a mensagem: "Até a próxima".
